An end-to-end AI agent system that scrapes job portals (LinkedIn, Indeed, Unstop), tailors resumes using local LLMs (Ollama), and manages applications via a Chrome extension and Next.js dashboard.

Machine Learning project for predicting Bangalore house prices. Features include data cleaning, feature engineering, and model evaluation using Linear Regression and Random Forest.
